packages.js

Establishes the webact object, the root object for all webact packages.  This file implements a very simple package system that is used to allow each package to create and use a separate namespace without requiring that all names be fully qualified.

This file must be loaded after jQuery and before any other webact packages.

The create function constructs a new object from a specified prototype.  This is similar to the create method from p22 of “JavaScript The Good Parts” by Douglas Crawford, O’Reilly, 2008.  This version lets the caller initialize members of the new object.  The initial parameter is optional.
